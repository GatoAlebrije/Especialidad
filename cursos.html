Especialidad en datos masivos e inteligencia de negocios.

Plan de estudio

Módulos

El plan de estudios de la especialidad está conformado por un total de 45 créditos, que serán cubiertos en 3 cuatrimestres (12 meses). El programa está conformado por 3 módulos que, se describen brevemente a continuación.

  • system_update_altMódulo 1: Adquisición de datos
    En este módulo se pretende que los estudiantes sean capaces de crear dispositivos de hardware, que realicen una determinada tarea. También que el estudiante sea capaz de leer los datos arrojados por el dispositivo y almacenarlos en alguna fuente de datos tales como: archivos de texto plano, base de datos entre otros.
  • settingsMódulo 2: Procesamiento de datos
    En este módulo se pretende que el estudiante conozca y aplique algunas técnicas existentes en la literatura para el preprocesamiento de datos (datos incompletos, inconsistentes, ruido, entre otros). También se pretende que el estudiante conozca y aplique algunas técnicas para la extracción de conocimiento con el fin de obtener un modelo de datos.
  • smartphoneMódulo 3: Aplicación
    En este módulo se pretende que el estudiante evalúe el modelo que obtuvo en el módulo anterior. La evaluación se hará con datos no considerados en la creación del modelo.
